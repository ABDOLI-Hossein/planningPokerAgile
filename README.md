# Application Planning Poker - Groupe G

Application de Planning Poker développée en Python (Tkinter) permettant d'estimer la difficulté de tâches en équipe.

## Fonctionnalités
- **Modes de jeu :** Strict (Unanimité) et Moyenne.
- **Interface Graphique :** Visualisation des cartes et des joueurs.
- **Persistance :** Chargement de backlog JSON et sauvegarde automatique (pause café).
- **CI/CD :** Tests unitaires et documentation automatique via GitHub Actions.

## Installation

   cd <NOM_DU_DOSSIER>
